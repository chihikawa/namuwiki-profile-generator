미리보기 — 실시간 반영
사용 가이드
📋 Tistory 복사
  1. 왼쪽 편집 탭에서 캐릭터 정보 입력
  2. 상단 📋 Tistory 복사 버튼 클릭
  3. Tistory 글쓰기 → 우측 상단 HTML 버튼 클릭
  4. 원하는 위치에 복사한 코드 붙여넣기
  5. 다시 기본모드로 전환 후 발행
💡 이미지는 Tistory에 직접 업로드 후 발행된 글에서 이미지 우클릭 → 이미지 주소 복사해서 URL 칸에 넣으세요.
⬇️ HTML 저장 — URL로 공유하기
  1. 상단 ⬇️ HTML 저장 버튼 클릭 → .html 파일 다운로드
  2. netlify.com/drop 접속
  3. 다운로드된 파일을 드래그앤드롭
  4. https://랜덤이름.netlify.app 형태의 URL 즉시 생성
  5. 이 URL을 SNS나 링크트리 등에 공유
💡 도메인을 구입해 연결하면 https://내도메인.com 으로도 접속할 수 있어요. Netlify에서 도메인 연결을 지원해요.
✍️ 본문 입력 문법
> 대사 → 대사 박스로 표시
[^각주내용] → 각주 번호 생성, 하단 미주에 자동 수집
